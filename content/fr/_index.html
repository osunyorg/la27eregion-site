Un laboratoire de transformation publique

header_text: >- La 27e Région header_cta: display: false label: >- Découvrir l'association target: "/a-propos" external: false contents_reading_time: seconds: 23 text: >- moins d'une minute contents: - kind: block template: pages title: >- slug: >- ranks: base: 2 children: 2 top: active: true title: value: >- À propos heading: 2 link: "/fr/a-propos/" description: >-

La 27e Région inspire et invente les services, les administrations et les modes d'action publics de demain. C'est un laboratoire de transformation publique, constitué en association indépendante, qui propose un espace pluridisciplinaire, réflexif et expérimental pour construire les futurs souhaitables de l'action publique.
